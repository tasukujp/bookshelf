# Bookshelf
Ruby on Rails の練習用に作成した本の管理Webアプリ

## Requirements
* ImageMagick
```
$ brew install imagemagick
```

## Environments
* Ruby 2.3.1
* Rails 4.2.7
* MySQL 5.7

## Usage
```
$ git@github.com:tasukujp/bookshelf.git
$ bundle install --path vendor/bundle
$ ./bin/rake db:create db:migrate db:seed
$ bundle exec rails s
```
